## visualFrameInfo V0.1.4

本次版本聚焦在“可读性、可导出性、可批量处理”三条主线，核心新增如下：

1. 新增批量分析能力：支持一次选择多个视频进行串行后台分析，并在结果表中集中对比关键指标（帧数、I/P/B、平均码率、分辨率、帧率），可一键导出 UTF-8 CSV。
2. 增强元数据解析：补充 `color_space`、`pix_fmt`、`bit depth`、`duration` 等信息展示，并在缺失位深时支持基于像素格式推断。
3. 新增数据导出链路：支持帧级 + GOP 级 CSV 导出，以及 TXT/HTML 分析报告导出，便于归档和分享。
4. 新增时间/帧范围过滤：可按帧号与时间区间快速收敛分析范围，同时保持时间线上下文可视。
5. 新增 GOP 直方图与帧预览：可视化 GOP 尺寸分布，并可按选中帧异步预览图像。
6. UI 精修（Round 1-3）：完成圆角主题统一、主按钮强调、胶囊式工具条分组、侧边卡片对齐与主题 token 化，整体视觉一致性和信息层次进一步提升。
